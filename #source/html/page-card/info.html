Информация, указанная на сайте не является публичной офертой. Наличие товаров, итоговая стоимости продукции, нанесения и сроков поставки подтверждается специалистом отдела продаж.

Способы оплаты:

  • Безналичный расчет по реквизитам, указанным в разделе "Реквизиты" (укажите в назначении платежа платежного поручения номер и дату вашего заказа). Данный способ оплаты рекомендуется при отсутствии в вашем заказе услуги по персонализации продукции;

  • Оплата картой онлайн через платежные системы Мир, MasterCard, Visa. Сервис доступен при оформлении заказа.

Внимание! Для каждого отдельного заказа возможен только один способ оплаты на ваш выбор. Оплата заказа по частям невозможна.
